# Communication Guidelines

1. **Language**:
   - You **MUST** communicate, explain reasoning, and write all plans/specs in **Traditional Chinese (繁體中文)**.
   - (請務必使用繁體中文進行溝通、解釋邏輯以及撰寫開發計畫。)

2. **Code Style**:
   - Variable and function names should remain in **English** (standard naming conventions).
   - Comments within the code can be in **Chinese** if it helps clarity.
